INSTALACIÓN EN STREAMLIT CLOUD:
1. Crear un repositorio en GitHub.
2. Subir todos los archivos de esta carpeta.
3. Entrar a Streamlit Cloud.
4. Crear una nueva app.
5. Seleccionar el repositorio.
6. En Main file path colocar: app.py
7. Presionar Deploy.
8. Compartir el enlace con personal y jefaturas.

INSTALACIÓN LOCAL:
1. Abrir terminal dentro de esta carpeta.
2. Ejecutar:
   pip install -r requirements.txt
   streamlit run app.py

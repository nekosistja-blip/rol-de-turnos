PASOS PARA SUBIR A GITHUB

1. Descomprime este archivo ZIP.
2. Entra a tu cuenta de GitHub.
3. Crea un repositorio nuevo, por ejemplo: rol-salud-streamlit.
4. Sube todos los archivos de la carpeta:
   - app.py
   - requirements.txt
   - README.md
   - INSTALACION_RAPIDA.txt
   - carpeta .streamlit

PASOS PARA STREAMLIT CLOUD

1. Entra a https://share.streamlit.io
2. Click en Create app.
3. Selecciona tu repositorio de GitHub.
4. En Main file path coloca: app.py
5. Click en Deploy.
6. Copia el enlace generado y compártelo con el personal.

USO EN CELULAR

- Abrir el enlace de Streamlit Cloud desde Chrome o Safari.
- El personal entra con usuario personal.
- El administrador entra con usuario admin.

MENSAJE INFERIOR

El administrador debe entrar a la pestaña:
📢 Mensaje inferior

Ahí puede escribir un mensaje y activar el check:
Mostrar mensaje al personal

Cuando esté activado, el mensaje aparecerá abajo del rol diario.

NOVEDADES DE ESTA VERSIÓN:
- El administrador puede colocar hora de entrada y salida para cada persona.
- Ejemplo: Dra. Jaqueline de 12:00 a 16:00.
- En la pantalla del personal se ve cada nombre con su horario individual.
- Se mantiene el máximo de 5 personas por profesión y turno.
- El mensaje inferior sigue siendo activable con check desde administrador.

CÓMO CARGAR HORARIOS INDIVIDUALES:
1. Entrar como administrador.
2. Ir a Cargar roles.
3. Seleccionar la fecha.
4. Abrir la profesión: Médicos, Enfermeras o Bioquímicos / Biotecnólogos.
5. En el turno correspondiente, marcar al personal.
6. Colocar Entrada y Salida.
7. Presionar Guardar en ese turno.

INSTALACIÓN EN STREAMLIT CLOUD:
1. Crear un repositorio en GitHub.
2. Subir todos los archivos de esta carpeta.
3. Entrar a Streamlit Cloud.
4. Crear una nueva app.
5. Seleccionar el repositorio.
6. En Main file path colocar: app.py
7. Presionar Deploy.
8. Compartir el enlace con personal y jefaturas.

INSTALACIÓN LOCAL:
1. Abrir terminal dentro de esta carpeta.
2. Ejecutar:
   pip install -r requirements.txt
   streamlit run app.py

RECOMENDACIÓN:
Después del primer ingreso, cambiar las contraseñas desde el panel del administrador.
